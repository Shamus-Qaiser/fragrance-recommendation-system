#  AI Fragrance Recommendation System

An intelligent full-stack application that recommends personalized fragrances based on user preferences using an interactive quiz-driven approach.



## Overview

This system helps users discover the perfect fragrance by analyzing their preferences through a structured quiz. Instead of generic suggestions, it delivers tailored recommendations based on individual taste profiles.

---

##  How It Works

1. User starts with a guided quiz
2. Selects preferences (e.g. fresh, woody, floral, strong, light)
3. System processes inputs
4. AI/logic matches preferences with fragrance dataset
5. Personalized recommendations are generated instantly

---

##  Features

* Quiz-based recommendation engine

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/your-username/fragrance-recommendation-system.git
cd fragrance-recommendation-system
```

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm start
```
